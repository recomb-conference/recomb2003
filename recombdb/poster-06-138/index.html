138. A Biological Information System for Homo sapiens: integrating expression, function, interaction and localisation

Dieter Maier[1] and Matthias Fellenberg1

Keywords: Biolgical Information System, gene expression, funcition, localisation

1. Introduction

In Computational Biology a long term goal clearly is the computation of simulations that represent system behaviour. The lever we need to apply to this end is the integration of qualitative models with quantitative data. Currently neither the models, due to a lack of sufficient qualitative understanding of biological processes on a system level, nor the required quantitative data exist. Creation of such models requires an iterative refinement going back and forth between model predictions and experimental results. As a first step a model will have to include all information that is available concerning a specific system, requiring a fixed point to anchor that information. In Geography software anchoring disperse information to one entity (the geographical localisation) is known as Geographic Information System. Accordingly the biological equivalent, using genes as anchor, has been labelled Biological Information Systems (BIS) by Endy and Brent (2002). We have developed a BIS which enables us to integrate all available gene centred information into one qualitative model (Fellenberg et al., 2002). The variety of information to be integrated is limited only by the current lack of computer readable information in the databases.

Here we present the application of our BIS to the human genome. We integrate information from ex-pression data with functional annotation regarding the biological process, molecular function and locali-sation of a gene product. Data integration allows us to compute multilevel correlations between different, independent experimental information. These multilevel correlations provide much stronger evidence for causal relation between genes than correlation based on single experimental approaches. Another impor-tant aspect of data integration is the discovery of new and unexpected dependencies, like the recent dis-covery of correlation between expression and genomic gene localisation (Spellman and Rubin, 2002). Using all available information allows us to reduce the complexity of a system by selecting specific subsets. For these subsets we are already able to compute qualitative system behaviour e.g. regarding active metabolic pathways of an organism. In this way we can start the iterative process of qualitative model development on a simplified system level.

2. Results

We have applied our method to the human expression data set of Iyer et al. (1999) and integrated it with the manual annotation of the Human Genome provided by Biomax Informatics AG (www.biomax.de). Expression of 9495 genes has been measured at 12 time points during human fibroblast serum induced differentiation, using cDNA microarrays.  The expression data are clustered using a neuronal network approach, the Self organising map (SOM) (Kohonen et al. 1995). From all currently available clustering algorithms the SOM is especially well suited to our approach as the topology of the data is conserved to a certain degree. Every clustering algorithm tries to group similar expression profiles into one cluster. However, similar profiles will sometimes be split artificially due to the arbitrary nature of clustering. Using a SOM similar expression profiles will still end up in neighbouring clusters, allowing us to calculate cluster similarities and correlate these with functional information.

SOM clustering requires the heuristic predetermination of the required cluster number to which the multidimensional data space is mapped by the neuronal network. We found that clusters averaging 20-30 expression profiles provide an optimal balance between splitting similar versus joining dissimilar expres-sion profiles. We therefore set the number of clusters to 400, which resulted in most of the clusters con-taining 20-25 expression profiles (mean: 25, min: 9, max: 56). From these clusters we can generate superclusters of similar expression profiles by calculating correlation coefficients for all clusters in rela-tion to a seed cluster.

Our method allows us to integrate other available data with the gene expression data as long as it is mapped to genes. Therefore we searched the integrated Human Genome database provided by Biomax with sequences retrieved from Genbank according to the EST and cDNA IDs provided by Iyers et al. From 9495 sequence IDs we retrieved 17 692 5´ and 3´ sequences which were searched against the 39,882 ORFs of the Biomax Human Genome Database. This allowed us to map the sequences from the Iyers data to annotated human genes, enabling the integration of the functional information with the expression data.

The first step of integration is the identification of groups of clusters which contain a significant number of genes with the same functional annotation. As mentioned above a functional annotation can be a defined biological process (e.g. cell cycle), a metabolic function (e.g. kinase), a metabolic process (e.g. amino acid synthesis), a localisation (e.g. mitochondria), a phenotype (e.g. oncogene) or any other defined, computer readable description. Once a set of clusters sharing genes of the same functional annotation has been identified the significance of that group is calculated. If genes without functional annotation are co-expressed within significant groups of specific function they can tentatively be labelled with the same process and become primary targets for experimental validation. If several compatible functional annotations (e.g. cell cycle, oncogene, nucleus) group a gene with unknown function to the same expression clusters the probability for association grows accordingly.  At the same time, where groups from different functional annotations overlap, the intersection may represent genes joining different biological programs.

From the correlated superclusters or the functional groups of clusters computed as described above we are able to develop qualitative models for co-expressed parts of the complete system. In this way we avoid the overt complexity of modelling all possible interactions and at the same time choose a meaningful subset for modelling, avoiding arbitrary partition.

Here we present an analysis of co-expressed genes for metabolic processes. We construct possible meta-bolic pathways dynamically by linking proteins according to common metabolites of their respective en-zymatic reaction. These dynamic pathways are then  compared to static pathways constructed from the literature. In this way we are able to define biologically meaningful reaction subsets from the large space of theoretically possible reactions. In addition the available genomic information can be checked against the biochemical requirements allowing the specific search for unidentified enzymes.

From the data integration and modelling presented above we can build refined theories available to experimental evaluation. The new data from these experiments will allow us to refine our models and expand the space of available data for integration. Once our models are sophisticated enough to qualitatively represent system behaviour and more quantitative data becomes available, the creation of quantitative simulations will become possible.
