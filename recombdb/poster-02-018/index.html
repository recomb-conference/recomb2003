18.  Consensus Homology Analysis and SEarch (CHASE)

Intikhab Alam[1] , Georg Fuellen2

Abstract

A number of homology search methods enable the search for members of a protein family. However, it is difficult to decide which method should be used if one wants to find as many true members of a protein family as possible. To overcome this problem we devised a general scheme to combine homology search methods. This scheme makes use of evidence such as Evalues, available from different homology search methods. On the basis of this scheme we develop a consensus homology search tool, called CHASE (Consensus Homology Analysis and SEarch). To generate consensus results, at present CHASE combines results from motif based methods such as Phi-blast, HMMsearch and Vmatch. We run chase for protein families in SWISSPROT and SCOP. Our results show that it is better to use CHASE if it is not known which method will perform well. Currently we are devising an objective procedure for evaluating both the methods embedded in CHASE and CHASE itself.
