117.  Consensus approach for benchmarking domain assignment in protein structures

Stella Veretnik, Nickolai Alexandrov, Phil Bourne, and Ilya Shindyalov

Keywords: protein structure, domain assignment, algorithm benchmarking

Introduction

Assignment of domains in protein structures remains to be a difficult and unsolved problem. Existing algorithms frequently produce contradicting results, different from expert’s assignment.  In this work we examine trends in the performance of a new generation of completely automated domain assignment algorithms. We have benchmarked the following automatic methods: DALI [3], DomainParser [4], and PDP [5] using the dataset of proteins with domains assigned by experts [6] and using the consensus subset of these proteins in which the domain definition agrees with two other expert-based assignments by SCOP [1] and CATH [2]..

Methods and Results

The benchmark was created from 2363 protein chains cut into domains by authors of the crystal structures [6].  The set was cleaned by purging chains that have more than 90% sequence identity, resulting in a set of 749 structures which was further reduced to 599 chains present in all manually curated databases (SCOP, CATH, and author’s assignments).  Using this benchmark we assessed the correctness of the number of domains assigned by each method as well as the consistency in the domain boundaries definition.

We first compared consistency of prediction of automatic and manual assignments.  We defined disagreement value as (variance of the number of domains  +1).  Disagreement values for each chain were calculated separately for automatic methods (DALI, DomainParser, PDP) and for manual methods (CATH, SCOP, author’s assignment).  We then looked at the ratio of disagreement value produced by automatic methods and the disagreement value produced by manual methods (for each chain).  We find that in 80% of the chains this ratio is 1, indicating that the domain assignment disagreement of automatic and manual methods is identical.  The remaining 20% of the chains are nearly evenly split between the cases in which manual disagreement is higher that automatic disagreement and visa versa.  We interpret this as an indication that for some proteins, consensus on domain assignment is as difficult to reach for human experts as for algorithms.[n1] 

Compared to the author’s assignment, most of the methods tend to under-cut (i.e. to assign too few domains), this tendency is particularly evident in SCOP (Figure 1).  The exception to this is automatic assignment method DALI, which produces equal fraction of over-cuts and undercuts.  For all methods the correct assignment rate is between 80 and 90%, with lowest for DALI (automatic assignment) and highest for CATH (manual assignment).

Domain assignment can be done differently even by experts. To avoid ambiguity in domain definition we have eliminated problematic cases from the benchmark and focused on the 84% of the chains (total of 502 chains), in which manual assignment methods reached the consensus.  When chains without manual consensus were removed, all automatic methods showed a rather dramatic drop in the fraction of incorrectly predicted under-cut assignments; while the effect on incorrectly predicted over-cuts was rather modest  (Figure 2).

Figure 2.  The effect of removal of chains with no manual consensus.  Gray bars: fraction of mis-assigned chains in the entire set ( 599 chains), back bars: fraction of mis-assigned chains in 502 chains with manual consensus.  Left: total mis-assignments, middle: under-cut mis-assignments, right: over-cut mis-assignments.

 


Automatic assignments were grouped by the number of chains which disagree with manual consensus.  We find that all automatic methods tested agreed with manual consensus for 443 chains; this represent 88% of all chains for which manual consensus was reached.  In the remaining cases (59 chains) one or more automatic method disagreed with manually assigned consensus.  We are studying these cases in further detail in an attempt to understand the underlying assumptions that lead to erroneous automatic assignments.  Some interesting cases are those in which automatic methods produced 3 different assignments (in 3 cases, one example in Figure 3).   Other interesting cases are those where all automatic methods failed to cut a chain, which is predicted by manual consensus to have 2 or 3 domains (3 such cases).  For all three cases failure to cut is associated with short DNA-binding proteins.
