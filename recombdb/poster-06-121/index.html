121. Integrative Gene Expression Data Analysis using the Optimal sub-vector method

R.Balasubramaniyan [1], J.Kaemper 1, H.W.Mewes [2] and S.Tornow 2

1    Summary

DNA chip technology enables the study of gene expression in large scale. Up to now, one of the main challenges has been to devise methods to group genes that show similar expression profiles. Most methods developed so far use the data from entire experimental conditions as a measure to determine the co-regulation of genes. However to study the transcriptional network of the genes, we should analyse the co-expressed genes from independent experimental conditions (1). Inspired by the latter we have devised a method called optimal sub-vector method.

We have developed an algorithm that calculates the optimal sub-vector between two gene vectors using the modification of the Dynamic Programming algorithm. This sub-vector can be compared to that of the method used in the BLAST algorithm to calculate the Highest Scoring Pattern (HSP) between two DNA or Protein sequences. We are now using the optimal sub-vector method to find the highest scoring alignment between two gene vectors. Using this method we could calculate precisely the time shifted direct relationships and time shifted inverted relationships also.  The output of the algorithm enables us to calculate the number of co-regulated genes under particular experimental conditions. This method differentiates from Local alignment Method (2) by predicting the optimal sub-vector between two gene vectors from all experimental conditions.

Using this method we relate Protein-Protein Interaction to Expression data under each experimental condition evaluating a score for the co-expression of substructure of the Protein-Protein Interaction network in each experimental condition.
