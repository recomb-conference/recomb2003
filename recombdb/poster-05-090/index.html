90. Predicting Protein Class with
Support Vector Machines


James Casbon1, Mansoor Saqi



Keywords: Protein Structure, Protein Class, Support Vector Machines

Introduction

Protein or domain class prediction has been attempted in a number of ways. Most of these attempt to predict class of a protein based only on a twenty-dimensional vector of amino acid frequencies. Despite the basic information used, success rates have been encouraging[1]. We are examining improving prediction accuracy using the support vector machine method[4] and present here some preliminary results concerning choice of dataset and the the effect of the inclusion of domain length.

Methods

We used the SVM-light library for the SVM calculations[2]. The multiple class prediction problems were converted to binary problems using the one-against-others approach.

Choice of Dataset

Previous prediction attempts using support vector machines have a test set put forward by Zhou[5] and performed a cross validation experiment to determine accuracy on the test set. We were interested in how the similarity of domains within the dataset would affect the accuracy of prediction. The SCOP database[3] was ideal for this task. Not only does it contain a large amount of hand curated information on domains and their structural classes, but it also provides datasets with differing levels of maximum pairwise similarity. These datasets can be found at http://astral.scop.edu.

We performed a ten-fold cross-validation on the SCOP 1.61 database using sets of sequences with (a) less than 20% sequence identity and (b) less than 95% sequence identity. The results can be seen in table 1. As can be seen in the table, using the 95% dataset results in a 7% increase in performance. Interestingly, the rate of correct prediction for a+b domains is significantly worse than for the other classes, and at 26% for the 20% set, little better than random.

Effect of Length

The effect of adding in the domain length to the vector presented to the support vector machine can also be seen in table 1. The overall improvement in accuracy is slight, but this masks a large improvement in prediction of a+b domains.
Table: Accuracy of Class Prediction with Different Datasets
a b a/b a+b overall
Scop 1.61 less than 20% identity 0.61 0.60 0.67 0.26 0.53
Scop 1.61 less than 20% identity with length 0.61 0.58 0.64 0.38 0.55
Scop 1.61 less than 95% identity 0.65 0.67 0.69 0.38 0.60
