E-mail: tt duembgen@stat.unibe.ch endcenter par small par vspace.3cm par noindentbf Keywords: feature selection, cDNA arrays, singular value decomposition par sectionlarge Introduction Singular value decomposition (SVD) citeABB00 and principal component analysis (PCA) citeHT00 have been used to identify clusters of genes in an unsupervised manner and using a single principle component at a time--in case of PCA always the first one. par We apply SVD to gene expression data for supervised feature selection, i.e. to identify deregulated genes and use linear combinations of the singular components instead of looking at only one singular component. Classical statistical approaches (e.g. p-values from Student's t-test) to this problem suffer from the dimensionality (lqqrqq), which is inherent to gene expression data and implies small group sizes and a multiple testing situation. We show that our method is more specific than the t-test under these circumstances. Moreover, it can be applied to experiments, which comprise more than two groups. par sectionlarge Methods We presume that the expression data has been normalized using any suitable method and is given in matrix form ( rows representing the genes and columns representing the arrays). Furthermore, there are mutually exclusive groups. The algorithm works as follows: par renewedcommandtheenumiromanenumi renewedcommandlabelenumi(theenumi ) beginenumerate item Transform the rows of the data matrix to have a mean of zero and perform SVD to the centered matrix. item Define a base vector of length for each of the groups (or any set of groups), using a centered indicator function. item For each base vector find a linear combination of the first right singular vectors (emphright linear combination, RLC), so that the angle between this linear combination and the base vector becomes minimal. item Use the same coefficients to evaluate the linear combination of the first left singular vectors (emphleft linear combination, LLC). endenumerate par Usually, a single (especially the first) component is not sufficient. That is why we use a linear combination of the first components. Whether is large enough, can be controlled by looking at the correlation between the RLC and the according base vector. We can see, that for the first experiment the second right singular component alone is very dominant--indeed, it is sufficient to discriminate between the groups. But for the second experiment things look different.
