60. A Unified Framework for Motif Finding: Finding Position Specific Scoring Matrices using Patterns with Mismatches


Eleazar Eskin 1

Keywords: Motif Finding, Position Specific Scoring Matrices, Patterns with Mismatches.



An important part of deciphering gene regulatory mechanisms is discovering transcription factor binding sites. These sites can be detected because they are typically over represented in genomic sequences. The detection of the over represented signals in sequences, or motif-finding has become a central problem in computational biology with vast literature.

There are two major computational frameworks for attacking the motif finding problem which differ in their representation of the patterns. The most widely used is the PSSM (Position Specific Scoring Matrix) representation. The goal of these algorithms is to obtain probabilistic representations of the over represented signals. These algorithms view the representation of the signals as continuous. Each position in a PSSM consists of a vector of probabilities for observing each nucleotide. The algorithms in this framework typically attempt to discover PSSMs that maximize a criteria such as the information content score. Popular approaches within this framework include MEME [1], CONSENSUS [4], Gibbs sampler [6], and AlignAce [5]. A major advantage of this framework is the fact that PSSMs are very powerful representation of the signals. A major disadvantage is that the problem of discovering the ``best'' PSSM is inherently very difficult. For a signal of length , the problem reduces to finding the global maximum of a non-convex continuous dimensional function. Typically algorithms in this framework can be characterized as either performing stochastic optimization or greedy searches and give no guarantees of finding the best PSSM or even PSSMs close to the best PSSMs.

The second framework uses the pattern with mismatches representation. These algorithms define a signal to be a consensus pattern and allow up to a certain amount of mismatches to occur in each instance of the pattern. The goal of these algorithms is to recover the consensus pattern with the highest number of instances. These approaches view the representation of the signals as discrete. The major advantage of many of these algorithms is that even the classical algorithms such as the 1984 Waterman et al. [9] and more efficient recent algorithms for finding patterns with mismatches such as the algorithms of Sagot [7] and MITRA [3] guarantee to find the best signals In addition, some pattern driven algorithms such as MITRA [3] are modified to guarantee to find the highest scoring signals for a given scoring function. However, a major disadvantage to the pattern with mismatch framework is the representation of the signals themselves. In general, the PSSM representation is much more expressive than the pattern with mismatches representations. For example, patterns with mismatches in general can not distinguish between conserved and unconserved positions.

The main difference between these two frameworks can be summarized by noting that the continuous PSSM representation is more expressive at the cost of the guaranteed convergence properties of the less expressive discrete pattern with mismatches representation. There is debate about which framework performs better [2,8]. Ideally, the best of both worlds approach would combine the expressiveness of the PSSM representation with the guarantees of convergence of the pattern with mismatches approach.

In this poster, we present MITRA-PSSM, an algorithm for discovering PSSMs for over represented signals in sequences which provides similar convergence guarantees to the pattern with mismatches approach. The key to our approach is formulating a common framework for pattern discovery which encompasses both the PSSM and pattern with mismatches representations. In this framework, we show how some PSSMs are equivalent to patterns with mismatches. Using this common framework, we are able to define an algorithm that functions on discrete patterns with mismatches while performing the analogous search over the space of PSSMs.

The MITRA-PSSM algorithm consists of two main parts. The first part eliminates the portions of the space of all PSSMs which cannot score higher than the threshold. The second part performs a local search over the remaining regions of the space of PSSMs. The main ideas behind the first part of the MITRA-PSSM algorithm is that we search for the best PSSMs by partitioning the space of all PSSMs into disjoint subspaces with each subspace assigned a corresponding pattern with mismatches. The space of all PSSMs is partitioned in a way that the ``relevant'' instances of the PSSM in the sample are instances of the pattern with mismatches. For each subspace of PSSMs and its corresponding pattern with mismatches we can obtain an upper bound on the score of all PSSMs in the subspace. If this maximum score is below our minimum score threshold we can eliminate all of the PSSMs from this subspace from consideration. After this phase of the algorithm, we are left with subspaces of the PSSMs in which we perform a local search for the highest scoring PSSMs. The advantage of the MITRA-PSSM algorithm as opposed to other algorithms is that we can obtain some guarantees about the PSSMs that the algorithm finds. The algorithm can be used to find multiple high scoring PSSMs, not just the highest scoring. This eliminates the need to mask high scoring signals to discover the remaining signals.
