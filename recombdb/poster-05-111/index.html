111.  3D-Hit, Fast Structural Comparison of Proteins

Dariusz Plewczyński[1], Jakub Paś[2], Marcin von Grotthuss2, Leszek Rychlewski2

Keywords: 3D-Hit, Protein Structure Comparison, Protein Structure Hashing, LiveBench, ToolShop

Short Structured Abstract

 

Summary:

3D-Hit is a fast scanning method for detecting structural similarities between proteins. The algorithm is based on a hashing function, which decomposes proteins into segments of 13 residues. The scanning procedures start with assigning a set of similar segments from database to each segment in the query protein. These initial hits are expanded by two iterations of structural superposition of larger segments of 99 and 299 residues. The method generates an alignment for the query protein by concatenating partial structural alignments.

 

Availability:

3D-Hit is available as an http server at http://BioInfo.PL/3D-Hit/ or as an e-mail server at 3dhit@cyborg.bioinfo.pl. The best (user-friendly) access to the server is through the protein structure analysis service at http://BioInfo.PL/Meta/pdb-test.pl.The program will be also incorporated into the Biodesigner package (http://www.pirx.com/biodesigner/).

Contact:

Dariusz Plewczynski (darman@icm.edu.pl)

 

Supplementary information:

 

3D-Hit was compared with DALI using the ToolShop server (http://BioInfo.PL/ToolShop/, dataset: 4). 3D-Hit is coupled to the continuous online server evaluation program LiveBench (http://BioInfo.PL/LiveBench/), as a reference server.

 

Text:

Three dimensional structure of proteins is highly conserved [1] during the evolution. The comparison between 3D structures makes it possible to establish distant relationships, even between protein families distinct in terms of sequence comparison alone. That is why the structural alignment of proteins increase our understanding of more distant evolutionary relationships [2]. The link between structural classification and sequence families enable us to study functions of various protein groups or fold classes (see the CATH or SCOP structural classification of proteins). Structural comparison is also extensively used in the experimental structure determination process for detecting potential tracing shifts in low-resolution models [3]. Intermediate conformations in folding simulation can be quickly compared with known folds to find structural classification of proteins.

Our search procedure enables the complete alignment of protein pairs. This algorithm is able to find rapidly all structurally related cousins for a query protein within templates database. Post-processing of the structural alignment can be conducted manually or using other, slower, but more accurate superposition methods.

Our structure comparison server uses a database of 5000 proteins collected from the PDB and clustered at 90% identity level. The database update is repeated every week to accommodate new structures appearing in the PDB.

The server was coupled to the ToolShop [4] structure prediction evaluation program, which compares protein structure prediction servers. The direct comparison of the DALI server with our server was done on the evaluation set including 100 query proteins (easy and hard targets). The number of correct models generated by both servers for easy targets was very similar, however the final alignment quality was better in the case of the DALI server. In case of hard targets with the evaluation method used based on MaxSub [5], our server gets better ratings in all categories, especially in the specificity analysis. On the contrary different results are in the case of two other evaluation methods: Touch [6] and LGscore [7].

We conclude that structural comparison based on segments similarity is comparable with the DALI server and represents an interesting complement to current structure comparison programs. Our algorithm is fast - it takes 1 minute standard PC (2 GHz) to search for structural homologous for a query protein of 200aa in a database of 5000 templates.
