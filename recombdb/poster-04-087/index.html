87. An Automated Analysis of Almost All Available ABC Proteins


Michael Spitzer, 1 Stefan Lorkowski, 2 Paul Cullen, 3 Georg Fuellen 4



Keywords: ABC proteins, homology search, phylogeny

Abstract

RiPE (Rigorous Phylogeny Environment) is an automatic procedure to perform evolutionary analyses of protein families using only homologous protein sequence data. Using RiPE, we assembled a set of 1,000 ABC protein sequences from model organisms, and subjected these to phylogenetic analyses.
We searched the proteomes of Homo, Drosophila, Arabidopsis, Encephalitozoon, Saccharomyces, Mus, Rattus, Fugu, Danio, Caenorhabditis, Anopheles, Schizosaccharomyces, Oryza, Guillardia, the archaeon Methanococcus jannaschii and three representative bacteria (Escherichia coli, Synechocystis sp., Mycoplasma genitalium).
The multiple alignment was done using ClustalW [1]; the overall tree was computed by neighbor joining via "quicktree" [2]. In the resulting phylogenetic tree, sequences known to belong to the same subfamily were placed into the same subtree, even if bootstrap values were very low. More specifically, taking the human subfamily designations, we can reproduce classifications from the literature in the case of Drosophila, Arabidopsis and Encephalitozoon. This reproduction was correct in almost all cases. In turn, we can place high confidence in the subfamily classification labels that we inferred for proteins of model organisms for which no classification has  yet been published as of today. Based on these phylogenetic analyses, we discuss possible naming schemes for ABC protein inventories of the model organisms that have not been classified so far.
